## Description

GraphQL and NestJs project.

## Installation

1. Clone proyect from github.

2. Copy `env.template` y rename to `env`

3. execute

```bash
$ npm install
```

4. Up docker-compose

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Seed

5. Ejecutar la "mutation" executeSeed, para poblar la DB.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
